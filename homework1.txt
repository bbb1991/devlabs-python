1. Написать программу высчитывания чисел фибоначчи. Пользователь вводит число n. Необходимо вычислить n-ное число Фибоначчи.
Числа фибоначчи - числа, которые равны двум предыдущим. Подробнее в википедий
К примеру 0, 1, 1, 2, 3, 5, 8, 13, 21, ...
Пример:
    Вход:
        10
    Вывод:
        34

2. Написать программу, вычисляющую площадь треугольника по переданным длинам трёх его сторон по формуле Герона.
Пользователь вводит a, b, c. Необходимо вывести S
Пример:
    Вход:
        3
        6
        8
    Вывод:
        7.644442425710328

3. Напишите программу, принимающую на вход целое число, которая выводит True, если переданное значение попадает в
интервал (−15,12]∪(14,17)∪[19,+∞) и False в противном случае.
Пример:
    Вход:
        15
    Вывод:
        True

    Вход:
        -56
    Выход:
        False

4. Напишите простой калькулятор.
		Поддерживаемые операции: +, -, /, *, mod, pow, div, где
		mod — это взятие остатка от деления,
		pow — возведение в степень,
		div — целочисленное деление.
Пример:
    Вход:
        1
        2
        +
    Вывод:
        3

    Вход:
        2
        3
        pow
    Вывод:
        8

5. Напишите программу, которая получает на вход три целых числа,
по одному числу в строке, и выводит на консоль в три строки сначала максимальное,
потом минимальное, после чего оставшееся число.
Пример:
    Вход:
        14
        18
        81
    Выход:
        81
        14
        18

6. Напишите программу который проверяет, введенное число является ли счастливым.
Билет считается счастливым, если сумма первых трех цифр совпадает с суммой последних
трех цифр номера билета.
Пример:
    Вход:
        090234
    Вывод:
        "Счастливый"

    Вход:
        123333
    Вывод:
        "Обычный"

7. В конкурсе участвуют 2 команды. В первой команде а участников,
во второй команде b участников. Есть пирог для командв победителя.
Нужно заранее разрезать пирог таким образом, чтобы можно было раздать кусочки пирога любой команде,
выигравшей соревнование, при этом каждому участнику этой команды должно достаться одинаковое число
кусочков пирога. На вход подается число a, b. Нужно вывести число n.
Пример:
    Вход:
        7
        5
    Вывод:
        35

    Вывод:
